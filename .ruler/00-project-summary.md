- 미니멀한 디자인에 가장 단순한 그리드를 이용한 쇼핑몰 사이트
- 사이트에 흰색배경, 검정색 UI외에 그 어떤 색도 사용하지 않음
- 라우터가 없는 1페이지 웹사이트
- 페이지 구성은 헤더 / 메인 그리드 섹션 
- 해더의 좌측에는 네비게이션 버튼, 중앙에는 그리드 아이템의 유형을 결정하는 필터, 우측에는 장바구니 버튼 배치
  - 네비게이션 버튼은 동적으로 여러 기능을 담담함
    - 메인화면에서는 그리드의 colums 수를 조절하는 버튼, 플러스 모양 (+)
    - 위 기능은 2번까지 확대 가능, 이떄 백버튼 모양으로 변경(<)
    - 그리고 그리드상에 아이템이 선택되되어 확대화면으로 진입했을때 메인 화면으로 돌아가는 네비게이션 역할, 이떄 백버튼 모양으로 변경(<)

- 메인 그리드 영역은 다음과 같은 규칙이 필요함.
  - viewport를 모두 채우는 fluid UI layout
  - 기본적으로 mui의 grid를 응요해서 사용
  - 다만 필터링 및, size에 의해서 아이템의 크기와 위치가 바뀔떄, 스태틱한 변화가 아닌 transition을 이용한 동적인 이동과 크기 변경이 적용되야함.
  - 아이템을 클릭하면 해당 아이템이 화면 정웅앙에 오도록 그리드 전체가 확대
  - 이떄 나머지 아이템들은 화면에서 fade out
  - 화대 화면에서 중앙 이미지 좌우의 화살표로, 해당 아이템의 다른 각도, 모델 착용샷 등을 Loop형태로 슬라이드
  - 확대 화면에서 새로 스크롤을 한다면 그리드상에서 다음 아이템으로 넘어감. (세로 스크롤 트랜지션과 함께)
  - 확대 화면에서 돌아가려면 헤더의 백버튼을 사용
