# Component Interaction Principles

## Header
그리드를 조작하는 필터 원리
카테고리 버튼 클릭 시 해당하는 제품만 남기고 나머지는 숨김 처리. 그리드가 즉시 재배치되며 남은 아이템들이 빈 공간을 채움.

## DynamicGrid
숫자, 위치에 따른 그리드 배치 원리
열 개수에 따라 아이템이 좌→우, 위→아래 순서로 자동 정렬. 8열/6열/4열 모드에서 같은 아이템이 다른 행/열 위치를 차지하며 재배치.

## GridContainer
선택된 아이템에 따른 그리드 전체 이동 + 확대
클릭한 카드의 중심점을 화면 중앙으로 이동시키며 전체 그리드를 2배 확대. 카드 위치를 기준점으로 삼아 주변도 함께 당겨짐.

## ProductDetailView (CarouselMatrix)
2차원 그리드 이동을 위한 캐러셀 원리
가로 방향은 같은 제품의 여러 이미지를 슬라이드, 세로 방향은 다른 제품으로 넘어가며 이미지 위치 기억. 십자 방향 네비게이션으로 전체 매트릭스 탐색.
